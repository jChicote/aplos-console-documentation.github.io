AplosWindowRequest

AplosWindowRequest describes how a window should be created and configured. It is passed to AplosWindowManager.OpenWindow, which uses it to spawn the window's content, size it, and wire up its close behaviour. It is a plain class (namespace AplosConsole.Windows) that implements no interfaces of its own.
