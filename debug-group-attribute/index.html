DebugGroupAttribute

DebugGroupAttribute marks a class for discovery by the debug settings scanner, grouping its DebugSettingAttribute fields under one named group. It is a sealed Attribute (namespace AplosConsole.Settings.Attributes) restricted via [AttributeUsage(AttributeTargets.Class, AllowMultiple = false, Inherited = false)] to single, non-inherited use on classes, and implements no interfaces of its own.

DebugGroupAttribute

Parameters:

  • groupName (string) — The display name of this settings group in the JSON output.

Example

[DebugGroup("Graphics")]
public class GraphicsSettings
{
    [DebugSetting(Label = "Vsync")]
    public bool vsync;
}

Description: Creates the attribute with the group's display name. Throws ArgumentException if groupName is null, empty, or whitespace.
